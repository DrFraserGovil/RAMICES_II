\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class J\+S\+L\+::\+Argument$<$ T $>$}

A class which allows arbitrary template parameters to be read in as command-\/line arguments or from a configuraiton file using a Name/\+Value pair system. Upon construction, the \mbox{\hyperlink{classJSL_1_1Argument_a83ada5bfa412192f76dd4290f679defd}{Value}} parameter takes the default value (if provided), until it is overriden by a successful argument/\mbox{\hyperlink{classJSL_1_1ArgumentInterface_afa2d1f96c4971070d3de5824f297312f}{Trigger\+String}} match.
