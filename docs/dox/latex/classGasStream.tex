\doxysubsection{Detailed Description}
A gas stream is how a homogeneously-\/sourced set of gas gets moved around . They are created through Source\+Events -\/ such as CCSN or accretion events.

{\footnotesize\ttfamily bool Gas\+Stream\+::\+Needs\+Recomputing\hspace{0.3cm}{\ttfamily [private]}}



A flag used to prevent excessive recomputing of the internal masses. By design, if \mbox{\hyperlink{classGasStream_a65732759a83d7eb8bd23d7f944fe5217}{Absorb()}}, \mbox{\hyperlink{classGasStream_af8e82b7a64f1d8a1af93db0473de284f}{Deplete()}} etc. calls are avoided, the components of \mbox{\hyperlink{classGasStream_a0d15ba77815c9a96e4a6a3f4988c8384}{Hot}} and \mbox{\hyperlink{classGasStream_a08dbd72ea4f2239f285dea932fa83a67}{Cold}} are invariant, so the values of \mbox{\hyperlink{classGasStream_a5cdf9a3ce114e3ddb766fbd8e97c40d9}{Mass()}} are constant. Therefore if this flag is false, simply returns the last computed value of the mass value.
