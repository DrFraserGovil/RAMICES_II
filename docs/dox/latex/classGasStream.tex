\doxysubsection{Detailed Description}
A gas stream is how a homogeneously-\/sourced set of gas gets moved around . They are created through Source\+Events -\/ such as C\+C\+SN or accretion events.

A flag used to prevent excessive recomputing of the internal masses. By design, if \mbox{\hyperlink{classGasStream_a65732759a83d7eb8bd23d7f944fe5217}{Absorb()}}, \mbox{\hyperlink{classGasStream_af8e82b7a64f1d8a1af93db0473de284f}{Deplete()}} etc. calls are avoided, the components of \mbox{\hyperlink{classGasStream_a48c6a6e0bed770b63231f2c7364b3db0}{Hot}} and \mbox{\hyperlink{classGasStream_a7d9cb8da4073a28aa8fe38921c048a7a}{Cold}} are invariant, so the values of \mbox{\hyperlink{classGasStream_a5cdf9a3ce114e3ddb766fbd8e97c40d9}{Mass()}} are constant. Therefore if this flag is false, simply returns the last computed value of the mass value.
