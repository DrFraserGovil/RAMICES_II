\doxysubsection{Detailed Description}
A parent class for various unit testing frameworks. Provides a consistent A\+PI for interacting with the unit tests. The class is essentially meaningless on its own, but when coupled to a decently written subclass (i.\+e. \mbox{\hyperlink{classJSL__Testing_1_1StringTest}{J\+S\+L\+\_\+\+Testing\+::\+String\+Test}}), it becomes very useful.
